# SpliceML

## About the project
This repository contains the code and resources for our study on identifying clinically significant alternative splicing (AS) events associated with progression-free survival (PFS) in colorectal adenocarcinoma (COAD) patients using a robust machine learning framework.

### Datasets
Datasets are available as release (https://github.com/mitrofanova-lab/SpliceML/releases/tag/datasets)

### Analysis
- TCGA-COAD training and testing analysis - **``TCGA_COAD_analysis.R``** script
- AC-ICAM analysis - **``AC_ICAM_analysis.R``** script
